第一部分：開始遊戲

Go to the MakeCode Arcade editor and select New Project to begin creating a new game.

Once the editor loads, you will see a green on start block already placed in the editor Workspace.

We want the game to start with a Title Screen to let the player know what game they are playing. To do that, we go to Game in the Toolbox and under the Prompt category, drag a splash block into the on start block. Give your game a title and put in as the text for the splash block.
